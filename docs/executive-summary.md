# PortTask

## The Nerve Center for Maritime Operations

---

## What is PortTask?

PortTask is a next-generation maritime operations platform that unifies vessel management, port call coordination, vendor procurement, and stakeholder collaboration into a single, intelligent system.

**One platform. Every stakeholder. Complete visibility.**

---

## The Problem

Maritime operations today are fragmented:

| Pain Point | Impact |
|------------|--------|
| **Scattered communications** | Emails, WhatsApp, phone calls—no single source of truth |
| **Manual coordination** | Port calls managed in spreadsheets, prone to errors |
| **Vendor chaos** | RFQs via email, no price comparison, no performance tracking |
| **Zero visibility** | Customers and crew left in the dark |
| **Reactive operations** | Problems discovered too late to prevent |

**Result:** Delays, cost overruns, frustrated stakeholders, and operational risk.

## Platform Architecture

### Three Interconnected Portals

| Portal | Domain | Users | Purpose |
|--------|--------|-------|---------|
| **Main Platform** | `key.porttask.com` | Internal operators | Full operational control |
| **Operations Portal** | `portal.porttask.com` | Customers, crew, agents | Stakeholder collaboration |
| **Vendor Portal** | `vendor.porttask.com` | Service providers | RFQ response & fulfillment |

---

## Core Modules

### Main Platform (`key.porttask.com`)

| Module | Function |
|--------|----------|
| **Command** | Real-time situational awareness, incident management, unified communications |
| **Plan** | Workspaces, voyage planning, scheduling, weather & readiness |
| **Execute** | Port call management, RFQ procurement, vessel tracking, live operations |
| **Vendors** | Vendor network, verification, performance management |
| **Analyze** | Reports, usage tracking, predictions, recommendations, benchmarks |
| **Configure** | Settings, users, automation rules, integrations |

### Operations Portal (`portal.porttask.com`)

| Role | Capabilities |
|------|--------------|
| **Customer** | Fleet tracking, voyage visibility, spend monitoring, approvals |
| **Crew (Master)** | Port call management, service requests, agent coordination |
| **Agent** | Task management, document uploads, vessel communication |

*Role-based access: One login, permissions determine what you see.*

### Vendor Portal (`vendor.porttask.com`)

| Feature | Description |
|---------|-------------|
| **RFQ Response** | Receive requests, submit quotes, track win/loss |
| **Order Management** | Accept orders, update status, upload delivery docs |
| **Performance** | View ratings, compare to peers, improve service |
